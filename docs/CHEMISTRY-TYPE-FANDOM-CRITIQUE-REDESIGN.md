## 제1부. 팬덤 검증 비평 -- "왜 아저씨가 만든 것 같은가"

### 1.1 기존 6타입 다시 보기

```
공명형 (Resonance) - 같아서 편한
대비형 (Contrast) - 달라서 끌리는
성장형 (Growth) - 서로를 발전시키는
폭풍형 (Storm) - 함께면 폭발적인
치유형 (Healing) - 함께면 안식
수수께끼형 (Enigma) - 예측 불가의 매력
```

### 1.2 진짜 팬덤에서 CP를 분류하는 방식 -- 실증 데이터

#### AO3 태그 시스템 (글로벌 팬덤의 표준 분류)

AO3(Archive of Our Own)는 전 세계 팬덤의 사실상 표준 분류 체계다. 2025년 기준 가장 많이 사용되는 관계 트로프 태그들:

| 순위 | 트로프 태그 | 팬덤 느낌 | 기존 6타입과의 대응 |
|------|-----------|----------|------------------|
| 1 | **Fluff** (솜사탕) | 달달함, 행복 | 치유형과 비슷하지만 더 가볍고 밝음 |
| 2 | **Angst** (앙스트) | 아프고 힘든 감정 | **대응 없음** -- 기존 체계에 "아픈 케미"가 없다 |
| 3 | **Hurt/Comfort** (헛컴) | 아프지만 위로받는 | 치유형의 한 갈래이나 더 구체적 |
| 4 | **Slow Burn** (슬로우번) | 천천히 타오르는 | **대응 없음** -- 기존 체계에 "속도"가 없다 |
| 5 | **Mutual Pining** (서로 짝사랑) | 서로 좋은데 모르는 | **대응 없음** |
| 6 | **Friends to Lovers** | 친구에서 연인으로 | 성장형에 가깝지만 "서사"가 아닌 "성질"로 분류됨 |
| 7 | **Enemies to Lovers** | 적에서 연인으로 | 대비형에 가깝지만 역시 "서사"가 없음 |
| 8 | **Found Family** | 만들어진 가족 | **대응 없음** |
| 9 | **Fake Dating** | 가짜 연애 | **대응 없음** |
| 10 | **Forbidden Love** | 금지된 사랑 | 수수께끼형과 거리가 멀다 |

**핵심 발견: 10개 중 5개가 기존 6타입에 대응점이 없다.**

#### Ship Dynamic Meme 문화 (트위터/텀블러)

2019년부터 트위터에서 폭발적으로 퍼진 "Ship Dynamic Art Meme"에서 팬들이 실제로 사용하는 분류:

```
Grumpy x Sunshine (찡그린 애 x 햇살 같은 애)
Sun x Moon (태양 x 달)
Chaos x Order (혼돈 x 질서)
Touch-Starved x Affectionate (스킨십에 굶주린 애 x 애정 과다 애)
"I'd die for you" x "I'd kill for you"
Oblivious x Obviously in love (눈치 없는 애 x 티 나는 애)
Soft but deadly x Tough but soft (겉은 부드럽지만 무서운 x 겉은 강하지만 부드러운)
Sweaterboy x Absolute Nightmare (착한 스웨터보이 x 완전 미친놈)
```

**핵심 발견: 이들은 전부 "A x B" 포맷이다. 관계를 한 단어로 정의하지 않고, 두 성격의 대비/조합으로 정의한다.**

이것이 기존 "공명형", "대비형" 같은 한 단어 분류와 근본적으로 다른 점이다. 팬덤에서 CP를 말할 때는 항상 **두 축이 동시에 존재**한다.

#### 한국 팬덤 언어

한국 팬덤에서 CP를 묘사할 때 쓰는 실제 언어:

- **케미폭발** / **시너지 미쳤다** -- 감탄의 표현이지 분류가 아님
- **밀당** -- 관계의 패턴을 묘사
- **티키타카** -- 주고받기의 리듬감
- **찐친케미** -- 진짜 편한 관계
- **CP감** (씨피감) -- 커플처럼 보이는 느낌 자체
- **"이 둘 불법 아니야?"** -- 케미가 너무 좋을 때의 감탄
- **"케미 과다로 신고감"** -- 위와 동일한 과장 표현

**핵심 발견: 한국 팬덤은 CP를 "타입"으로 부르지 않는다. "분위기"와 "느낌"으로 말한다.**

"이 CP는 공명형이야"라고 말하는 팬은 없다.
"이 CP는 진짜 밀당 미쳤어" 또는 "이 둘 찐이야 찐"이라고 말한다.

#### 일본 동인 문화 (Pixiv/동인지)

일본 팬덤에서 CP 콘텐츠를 분류하는 태그 체계:

| 일본어 | 발음 | 의미 | 분위기 |
|--------|------|------|--------|
| 甘々 | 아마아마 | 달달함 | 설탕 녹는 듯한 행복 |
| ほのぼの | 호노보노 | 따뜻하고 잔잔한 | 이불 속 같은 안도감 |
| 切ない | 세츠나이 | 아련한, 가슴 아픈 | 비 오는 날 창밖 보는 느낌 |
| すれ違い | 스레치가이 | 엇갈림 | 같은 곳에 있는데 만나지 못하는 |
| 共依存 | 쿄이존 | 공의존 | 서로 없으면 안 되는 위험한 관계 |
| 溺愛 | 데키아이 | 맹목적 사랑 | 상대만 보이는 극단적 애정 |

**핵심 발견: 일본 팬덤의 분류는 "감정의 결"이다. 관계의 성질이 아니라 "그 관계를 볼 때 느끼는 감정"이 분류 기준이다.**

#### 중국 팬덤 언어 (磕CP 문화)

중국 팬덤의 CP 소비 문화에서 쓰이는 핵심 분류:

| 중국어 | 의미 | 팬 경험 |
|--------|------|---------|
| **糖** (탕/sugar) | 달달한 순간들 | "이 CP 설탕 쏟았어!" |
| **刀** (다오/knife) | 가슴에 칼 꽂는 앙스트 | "작가가 칼 들었어..." |
| **甜宠** (티엔총) | 달달하게 애지중지 | 꿀 떨어지는 연애 |
| **虐心** (위에신) | 심장을 쥐어짜는 | 눈물 흘리면서 보는 |
| **双向奔赴** (솽샹벤푸) | 쌍방향 질주 | 서로를 향해 달려가는 |
| **相爱相杀** (샹아이샹샤) | 사랑하면서 죽이는 | love-hate의 극치 |
| **欢喜冤家** (환시위안지아) | 기쁨의 원수 | 티격태격 bickering |

**핵심 발견: 중국 팬덤은 CP 경험을 "맛"으로 분류한다. 糖(설탕)과 刀(칼)라는 은유가 지배적이다. 이것은 향수 브랜드가 가져갈 수 있는 강력한 연결점이다.**

### 1.3 "아저씨가 만든 것 같은" 7가지 구체적 이유

위 조사를 바탕으로, 기존 6타입이 왜 팬덤 감성과 괴리되는지 구조적으로 분석한다.

#### 이유 1: 분류의 축 자체가 틀렸다

팬덤은 CP를 **"이야기의 패턴"(서사 트로프)**으로 분류한다:
- enemies to lovers (적 -> 연인)
- slow burn (천천히 타오르는)
- friends to lovers (친구 -> 연인)

기존 설계는 **"관계의 성질"(형용사)**로 분류한다:
- 공명형 (같은)
- 대비형 (다른)
- 성장형 (발전하는)

**차이**: 트로프는 "어디서 시작해서 어디로 가는가"라는 **방향성**이 있다. 형용사에는 방향성이 없다. 팬이 CP에 빠지는 이유는 "관계가 같아서"가 아니라 "적이었는데 사랑에 빠지는 그 과정"이 좋기 때문이다. 과정이 빠져 있으면 죽은 분류다.

#### 이유 2: 감정이 없다

"Resonance"에서 무슨 감정이 느껴지나? 물리학 용어다.

반면:
- "slow burn" -- 타오르는 이미지, 기다림의 고통, 터지는 순간의 카타르시스
- "hurt/comfort" -- 아픔과 위로가 교차하는 감정의 롤러코스터
- "糖" (설탕) -- 입 안에 퍼지는 단맛, 행복의 물리적 감각

팬덤 언어는 **감각적**이다. 맛, 온도, 촉감, 속도를 느낄 수 있어야 한다.

#### 이유 3: "A x B" 구조가 없다

Ship Dynamic Meme에서 보듯, 팬덤의 CP 분류는 항상 **두 극의 조합**이다:
- Grumpy **x** Sunshine
- Touch-Starved **x** Affectionate
- Chaos **x** Order

"대비형"이라는 한 단어는 이 **조합의 재미**를 다 날려버린다. "달라서 끌리는"이 아니라, **뭐와 뭐가 다른데, 그 다름이 어떤 재미를 만드는지**가 핵심이다.

#### 이유 4: 영어 타입명의 어색함

한국 팬덤 서비스에서:
- "Resonance" -- 물리 수업인가?
- "Enigma" -- 전쟁 영화 암호기인가?
- "Storm" -- 날씨 앱인가?

팬이 자기 CP 결과를 트위터에 올릴 때 **"우리 CP 스톰이래!"**라고 쓸까? 안 쓴다. **"우리 CP 향기 궁합이 '폭풍 전야의 정적'이래 ㄹㅇ 소름"**이라고 써야 공유가 일어난다.

#### 이유 5: 6이라는 숫자의 문제

6개 고정 타입은 MBTI의 열화판 느낌이다. 하지만 MBTI가 성공한 이유는 16개 조합이 충분히 세분화되어 "나만의 것" 같은 느낌을 주기 때문이다. 6개는 너무 적어서 "이건 나야!"라는 동일시가 약하고, 너무 많아서 "전부 비슷비슷"해 보인다.

팬덤에서의 CP 분류는:
- AO3: 수백 개의 태그가 유기적으로 조합됨
- 일본: 수십 개의 감정 태그가 중첩됨
- 중국: 糖/刀 같은 2-3개 극축 + 무한한 구체적 표현

**정답은 "적은 수의 넓은 축 + 무한한 동적 표현"이다.**

#### 이유 6: "타입"이라는 프레이밍 자체

"당신의 케미 타입은 공명형입니다" -- 이것은 진단서다.
"이 CP의 향기는 '새벽 3시의 옥상'이야" -- 이것은 이야기다.

팬은 진단받고 싶은 게 아니다. **이야기를 발견하고 싶다.**

MBTI가 "당신은 INTJ입니다"라고 할 때 성공하는 이유는, INTJ라는 라벨 뒤에 **풍부한 서사**(특성, 유명인, 궁합, 밈)가 있기 때문이다. "공명형"이라는 라벨 뒤에는 아무것도 없다. 팬덤의 역사도 없고, 밈도 없고, "나도 공명형!"이라고 공감할 커뮤니티도 없다.

#### 이유 7: 향수와의 연결이 인위적이다

"공명형이면 비슷한 계열의 향수 2개" -- 이것은 **논리적** 연결이지 **감성적** 연결이 아니다.

중국 팬덤의 糖/刀 체계를 보라:
- 糖 (달달한 CP) -> 구르망 계열의 달콤한 향 -> **자연스러운 감각적 대응**
- 刀 (앙스트 CP) -> 스모키/우디의 쓰라린 향 -> **자연스러운 감각적 대응**

향수와 트로프의 연결은 "논리"가 아니라 **"공감각(synaesthesia)"**이어야 한다. slow burn은 천천히 올라오는 인센스 향이고, enemies to lovers는 스파이시가 플로럴로 변하는 향의 전개다.

## 제2부. "타입을 정의하는 이유" 자체 재검토

### 2.1 타입 분류가 정말 필요한가?

세 가지 옵션을 비교한다.

#### 옵션 A: 팬덤 언어로 고정 타입 재설계
- 장점: 도감 수집, 통계, 커뮤니티 기능이 가능
- 단점: 아무리 잘 만들어도 "고정 타입"은 결국 반복되면 식상해짐
- 바이럴성: 중간 (처음엔 좋지만 "또 이거야" 피로도)

#### 옵션 B: 고정 타입 폐기, 완전 동적 생성
- 장점: 매번 유일무이한 결과, "나만의 것" 극대화
- 단점: 도감 수집 불가, 통계 불가, "나도 이거 나왔어!" 공감 불가
- 바이럴성: 낮음 (공유해도 상대가 같은 결과를 못 받으므로 대화가 안 됨)

#### 옵션 C: 하이브리드 (넓은 축 + 동적 칭호) -- **최종 추천안**
- 장점: 도감 수집 가능 + 매번 고유한 경험 + 커뮤니티 공감 가능
- 단점: 설계 복잡도 증가
- 바이럴성: **최고** -- "우리 CP 밀당향인데 칭호가 '비 오는 날 우산 하나'래 ㅋㅋㅋ 너무 맞아"

### 2.2 왜 하이브리드인가

**SNS 바이럴의 핵심 공식:**
```
공유 동기 = "공감할 수 있는 넓은 범주" + "나만의 고유한 디테일"
```

MBTI가 성공한 이유: "INTJ"라는 넓은 범주 (공감 가능) + "INTJ인 유명인 목록, INTJ 밈" (디테일)
퍼스널컬러가 성공한 이유: "가을 웜톤"이라는 넓은 범주 + "가을 웜톤에 어울리는 구체적 색상 팔레트" (디테일)

케미 타입도 같은 구조여야 한다:
- 넓은 축: "밀당향" (다른 사람도 "나도 밀당향!" 할 수 있음)
- 고유 칭호: "비 오는 날 우산 하나" (이 CP만의 것)
- 고유 향 서사: "A의 시트러스가 먼저 도망치고, B의 우디가 천천히 쫓아가는..." (이 CP만의 것)

## 제3부. 완전 재설계안

### 3.1 새로운 분류 체계: "케미향" (4대 향조 + 동적 칭호)

고정 타입을 6개에서 **4개로 줄이되**, 각각을 **향수의 향조(accord) 언어**와 **팬덤 트로프**의 교차점에 놓는다. 향수 브랜드 서비스이므로 향의 언어가 자연스럽게 녹아들어야 한다.

```
      [4대 케미향]

  1. 밀당향 (Push & Pull Accord)
     트로프: Enemies to Lovers, Tsundere, 相爱相杀, 欢喜冤家
     향 패턴: 상충하는 두 노트가 시간차로 교차
     감정: 긴장, 설렘, 짜릿함, "이 텐션 뭐야"
     색: 빨강 x 파랑 계열

  2. 슬로우번향 (Slow Burn Accord)
     트로프: Slow Burn, Mutual Pining, Friends to Lovers, すれ違い
     향 패턴: 베이스노트가 시간이 지나며 천천히 올라옴
     감정: 기다림, 아련함, 터지는 순간의 카타르시스
     색: 앰버 x 골드 계열

  3. 달달향 (Sugar Accord)
     트로프: Fluff, 甜宠, 溺愛, ほのぼの, Found Family
     향 패턴: 구르망+플로럴의 부드러운 하모니
     감정: 행복, 안도, 포근함, "세상 편안"
     색: 핑크 x 크림 계열

  4. 폭풍향 (Storm Accord)
     트로프: Angst, Hurt/Comfort, 虐心, 共依存, 切ない
     향 패턴: 강렬한 노트가 충돌 후 하나로 수렴
     감정: 격렬함, 눈물, 카타르시스, "아프니까 사랑이다"
     색: 보라 x 검정 계열
```

### 3.2 왜 이 4개인가 -- 실증적 근거

| 케미향 | AO3 대응 태그 | 일본 대응 | 중국 대응 | 한국 대응 |
|--------|-------------|----------|----------|----------|
| 밀당향 | Enemies to Lovers, Belligerent Sexual Tension | ツンデレ | 相爱相杀, 欢喜冤家 | 밀당, 티키타카 |
| 슬로우번향 | Slow Burn, Mutual Pining, Friends to Lovers | すれ違い, 切ない | 双向奔赴 | "언제 이어지냐" |
| 달달향 | Fluff, Domestic Fluff, Established Relationship | 甘々, ほのぼの | 糖, 甜宠 | 찐, 찐친케미, "달달" |
| 폭풍향 | Angst, Hurt/Comfort, Whump | 鬱, 重い | 刀, 虐心 | "칼각", "작가 미쳤냐" |

**4개 축이 한/일/중/글로벌 팬덤 모두를 관통한다.** 이것은 자의적 분류가 아니라, 전 세계 팬덤이 이미 사용하고 있는 분류를 향수 언어로 번역한 것이다.

또한 4개라는 숫자는:
- 도감 수집이 너무 쉽지도 어렵지도 않은 적정 수
- 서로 완전히 구별되어 "또 이거야" 피로가 없음
- 2x2 매트릭스로 시각화 가능 (긍정-부정 x 빠름-느림)

### 3.3 2x2 감정 매트릭스

```
        느린(Slow)          빠른(Fast)
      ┌─────────────┬─────────────┐
 달콤  │             │             │
(Sweet)│  슬로우번향   │   달달향     │
      │  기다림의 미학 │  즉각적 행복  │
      ├─────────────┼─────────────┤
 강렬  │             │             │
(Intense)│  폭풍향     │   밀당향     │
      │  깊은 감정의 파도│ 짜릿한 텐션  │
      └─────────────┴─────────────┘
```

이 매트릭스 자체가 공유 가능한 콘텐츠가 된다. "우리 CP는 느리고 강렬한 폭풍향이래" -- 2축 좌표가 CP의 위치를 직관적으로 보여준다.

### 3.4 동적 칭호 시스템 -- AI가 매번 고유하게 생성

4대 케미향은 넓은 범주일 뿐이다. 진짜 공유 가치는 **AI가 매번 생성하는 고유 칭호**에 있다.

#### 칭호 생성 규칙

```
[케미향 범주] + [고유 칭호] + [킬러 카피]

예시:
밀당향 -- "비 오는 날의 우산 한 개"
  "A의 시트러스가 먼저 도망치는데, B의 우디가 천천히 쫓아와서 결국 감싸버려.
   이 둘, 밀고 당기다가 결국 한 우산 아래야."

슬로우번향 -- "새벽 4시의 읽씹"
  "B의 머스크가 깊은 밤처럼 깔려 있는데, A의 플로럴이 아주 천천히,
   정말 천천히 스며들어. 읽었는데 답 안 하는 그 심장... 근데 결국 답이 오잖아."

달달향 -- "이불 밖은 위험해"
  "A의 바닐라와 B의 앰버가 만나면 세상에서 가장 따뜻한 담요가 돼.
   이 CP 향 맡으면 이불 밖으로 나가기 싫어져."

폭풍향 -- "상처 위의 반창고"
  "A의 스파이시가 날카롭게 공기를 가르면, B의 샌달우드가 그 자리를
   부드럽게 메워. 아프니까 사랑이고, 그래서 이 향이 눈물 나."
```

#### 칭호 생성 AI 프롬프트 가이드라인

```
[칭호 생성 규칙]

1. 반드시 한국어, 7-15자 이내
2. 구체적 상황이나 장면이 그려져야 함
   좋은 예: "새벽 4시의 읽씹", "비 오는 날의 우산 한 개"
   나쁜 예: "운명의 만남", "영혼의 파트너" (너무 추상적)
3. 일상적이되 시적인 표현
   좋은 예: "이불 밖은 위험해", "카페 구석 자리 둘"
   나쁜 예: "폭풍우 속의 나비" (너무 문학적)
4. SNS에서 보면 "ㅋㅋㅋ 이거 뭐야 너무 맞아"라고 반응할 수 있는 것
5. 캐릭터 특성(traits, scentCategories)에서 파생되어야 함
6. 절대 부정적이면 안 됨 -- 모든 칭호가 매력적이어야 함
```

### 3.5 각 케미향에 자연스러운 향 조합 패턴

향수 레이어링과 트로프의 연결은 **공감각적**이어야 한다. 논리적 매칭이 아니라, "이 트로프를 읽을 때 느끼는 감각"이 향으로 번역되어야 한다.

#### 밀당향 (Push & Pull Accord)

```
향 패턴: 상충 -> 교차 -> 합일
레이어링 서사: "처음에 A향과 B향이 따로 놀아. 서로 안 섞여.
  근데 시간이 지나면... 경계가 흐려지면서 둘만의 새로운 노트가 나타나"

전형적 조합:
  A: 시트러스/스파이시 (날카로운, 공격적)
  B: 우디/머스크 (깊은, 수용적)
  또는 반대

레이어링 순서: A 먼저 -> 3분 후 B (서로 다가가는 시간차)

트로프 연결:
  Enemies to Lovers = 스파이시(적대) -> 플로럴(사랑)로의 변환
  츤데레 = 시트러스(냉정한 첫인상) -> 구르망(숨겨진 달콤함)이 나중에 나옴
```

#### 슬로우번향 (Slow Burn Accord)

```
향 패턴: 미약한 시작 -> 점진적 강화 -> 폭발적 클라이맥스
레이어링 서사: "처음엔 거의 안 느껴져. 뭐야, 아무 일도 안 일어나나?
  싶은데... 30분 후에 갑자기 '어? 이거 언제부터 이랬어?' 하는 순간이 와"

전형적 조합:
  A: 가벼운 플로럴/프루티 (존재감 약한 시작)
  B: 깊은 앰버/우디/머스크 (시간이 지나야 올라오는)

레이어링 순서: B 먼저(보이지 않는 깊이) -> A 위에(눈에 보이는 표면)
  = B가 먼저 피부에 스며들고, A가 위를 덮지만, 시간이 지나면 B가 올라옴

트로프 연결:
  Slow Burn = 탑노트(첫만남)는 심심한데 베이스(깊은 관계)에서 폭발
  Mutual Pining = 두 향이 각자 피부 위에서 서로를 "찾는" 과정
  すれ違い = 향이 엇갈리다가 마침내 만나는 순간
```

#### 달달향 (Sugar Accord)

```
향 패턴: 즉각적 조화 -> 부드러운 유지 -> 잔향까지 달콤
레이어링 서사: "이 둘은 만나자마자 완벽해. 고민할 필요가 없어.
  A와 B가 만나면 세상에서 가장 편안한 향이 돼. 집 냄새 같은 거야."

전형적 조합:
  A: 구르망/바닐라/달콤한 플로럴
  B: 파우더리/앰버/소프트 우디

레이어링 순서: 동시에 (= 이 둘은 처음부터 함께)
  양 손목에 각각 뿌리고 비비면, 즉시 하나가 됨

트로프 연결:
  Fluff = 바닐라 + 파우더 = 솜사탕 질감의 행복
  Found Family = 따뜻한 머스크(안도감) + 부드러운 앰버(소속감)
  ほのぼの = 차(tea) 노트 + 꽃 향 = 봄날 오후의 평화
```

#### 폭풍향 (Storm Accord)

```
향 패턴: 강렬한 충돌 -> 혼란 -> 하나로 수렴하며 카타르시스
레이어링 서사: "처음에 두 향이 부딪혀. 편안하지 않아. 긴장되고, 심장이 빨라져.
  근데 그 충돌이 지나면... 어떤 향도 혼자서는 낼 수 없는 깊이가 나타나.
  아름다운 건 항상 고통 뒤에 있더라."

전형적 조합:
  A: 스모키/레더/스파이시 (날것의 강렬함)
  B: 인센스/프랑킨센스/깊은 우디 (정신적 깊이)

레이어링 순서: A를 강하게 -> B를 그 위에 겹쳐서 (충돌의 물리적 경험)
  두 향이 "싸우는" 5-10분이 이 CP의 핵심 경험

트로프 연결:
  Angst = 스모키노트(고통) -> 앰버(치유)로의 전환 과정
  Hurt/Comfort = A가 아픈 향(쓴, 스모키)이면 B는 감싸는 향(따뜻한 앰버, 바닐라)
  共依存 = 서로의 노트 없이는 완성되지 않는 조향 구조
```

### 3.6 팬이 SNS에 공유하고 싶은 결과 표현

#### 공유 카드 구성 요소 (우선순위)

```
1순위 (0.5초 안에 전달): 두 캐릭터 이미지 + 케미향 이름
2순위 (2초 안에 전달): 고유 칭호 (큰 글씨)
3순위 (5초 안에 전달): 킬러 카피 1줄
4순위 (관심 있으면): 향 조합 정보

[ 공유 카드 레이아웃 ]

+------------------------------------------+
|                                          |
|  [A 이미지]    x    [B 이미지]           |
|                                          |
|          ─── 밀당향 ───                  |
|                                          |
|    " 비 오는 날의 우산 한 개 "            |
|                                          |
|  "A의 시트러스가 도망치면                 |
|   B의 우디가 쫓아와서 결국 감싸버려"      |
|                                          |
|  AC'SCENT 07  x  AC'SCENT 14            |
|                                          |
|  #뿌덕스팟 #케미향 #밀당향               |
+------------------------------------------+
```

#### 트위터 자동 생성 텍스트

```
[TEMPLATE]

우리 {A이름} x {B이름} 케미향 결과:

{케미향 아이콘} {케미향 이름}
"{고유 칭호}"

{킬러카피}

#뿌덕스팟 #케미향분석 #{케미향이름}

[EXAMPLE]

우리 사토루 x 스구루 케미향 결과:

🌪️ 폭풍향
"신이 만든 균열"

"A의 오존 같은 투명함이 B의 인센스 같은 그림자를 만나면
세상이 갈라져. 이 CP의 향은 신화가 무너지는 냄새야."

#뿌덕스팟 #케미향분석 #폭풍향
```

이 포맷이 바이럴을 만드는 이유:
1. **케미향 이름**이 공감 포인트 (다른 팬도 "나도 폭풍향!" 가능)
2. **고유 칭호**가 유니크 포인트 (이 CP만의 것)
3. **킬러카피**가 감정 포인트 (읽는 사람도 "와 소름" 반응)
4. **해시태그**가 커뮤니티 포인트 (#폭풍향 검색하면 같은 유형 모아보기)

### 3.7 범용성 검증 -- 한/일/중/글로벌

| 케미향 | 한국어 | 일본어 | 중국어 | 영어 |
|--------|--------|--------|--------|------|
| 밀당향 | 밀당향 | 駆け引きの香り | 推拉香 | Push & Pull Accord |
| 슬로우번향 | 슬로우번향 | じんわり香 | 慢燃香 | Slow Burn Accord |
| 달달향 | 달달향 | あまあま香 | 甜蜜香 | Sugar Accord |
| 폭풍향 | 폭풍향 | 嵐の香り | 风暴香 | Storm Accord |

각 언어에서 자연스러운 이유:
- **한국어**: 팬덤 일상어 그 자체 (밀당, 달달, 슬로우번은 이미 쓰이는 단어)
- **일본어**: pixiv 태그 체계와 직접 대응 (甘々, 切ない 등)
- **중국어**: 磕CP 문화의 糖/刀 체계와 직접 대응
- **영어**: AO3 트로프 태그와 직접 대응 + "Accord"가 향수 용어

"Resonance"는 어느 언어 팬덤에서도 쓰이지 않는 단어였다. 새 체계는 **이미 팬들이 쓰고 있는 단어**를 가져온 것이다.

## 제4부. 기존 코드와의 연결점

### 4.1 기존 analysis.ts와의 호환

현재 `ImageAnalysisResult`의 `TraitScores` (10축)과 `ScentCategoryScores` (6축)가 케미향 분류의 입력이 된다:

```
케미향 판별 로직 (개념):

1. 두 캐릭터의 TraitScores 차이 벡터를 계산
2. ScentCategoryScores의 상호보완성/유사성을 분석
3. 4대 케미향 축에 매핑:

   밀당향 조건:
     - traits 차이가 큰 축이 3개 이상
     - scentCategories가 상반된 방향 (시트러스 vs 우디 등)
     - OR: 한쪽 darkness/charisma 높고 다른 쪽 cute/purity 높음

   슬로우번향 조건:
     - traits 유사도가 중간 (같지도 다르지도 않은)
     - scentCategories에 점진적 변화 패턴
     - OR: 한쪽이 freshness/purity 높고 다른 쪽이 elegance/luxury 높음

   달달향 조건:
     - traits 유사도가 높음
     - scentCategories 모두 부드러운 계열 (floral, musky, fruity)
     - OR: 양쪽 다 cute/purity/warmth 계열 높음

   폭풍향 조건:
     - traits 극단값이 많음 (7 이상인 것이 3개+)
     - scentCategories에 강렬한 노트 (spicy, woody 높음)
     - OR: 양쪽 다 darkness/charisma/sexy 높음
```

### 4.2 기존 CP-LAYERING-CRITIQUE-AND-REDESIGN.md의 8타입과의 관계

기존 문서에서 제안한 8타입:
```
explosive(폭발형), slow_burn(슬로우번), push_pull(밀당형), resonance(공명형),
plot_twist(반전형), shield(보호형), gravity(중력형), fusion(합체형)
```

이것을 4대 케미향으로 흡수:
```
밀당향 <- push_pull + plot_twist (긴장과 반전은 같은 축)
슬로우번향 <- slow_burn + gravity (천천히 끌려가는 것은 같은 축)
달달향 <- resonance + shield + fusion (편안함/보호/합일은 같은 축)
폭풍향 <- explosive (강렬함)
```

8개는 너무 세분화되어 있었다. 팬이 "보호형"과 "합체형"의 차이를 직관적으로 구분하기 어렵다. 4개면 충분하고, 세부 차이는 동적 칭호가 표현한다.

### 4.3 기존 메타포 체계와의 통합

기존 문서의 메타포 체계(시간축, 공간축, 감정축, 음악, 색채, 요리, 자연현상, 텍스타일, 건축, 게임)는 **그대로 유지하되**, 케미향별로 우선 메타포를 배정:

```
밀당향 -> 우선: 게임/전투, 요리(단짠), 자연현상(태양x달)
슬로우번향 -> 우선: 시간축, 공간축, 음악(크레센도)
달달향 -> 우선: 요리(달달한), 텍스타일(부드러운), 색채(따뜻한)
폭풍향 -> 우선: 자연현상(폭풍), 감정축, 건축(무너지고 다시 세우는)
```

## 제5부. 최종 비교 -- 구체계 vs 신체계

### Before (기존 6타입)

```
사용자가 결과를 받았을 때:

"당신의 케미 타입: 공명형 (Resonance)"
"같아서 편한 두 사람입니다."

사용자 반응: "...음... 그렇구나" (트위터에 공유하지 않음)
```

### After (새로운 4대 케미향 + 동적 칭호)

```
사용자가 결과를 받았을 때:

🌪️ 폭풍향
" 신이 만든 균열 "

"A의 오존 같은 투명함이 B의 인센스 같은 그림자와 부딪히면
 세상에 없던 향이 태어나. 이건 향이 아니라 신화야.
 무너지고 다시 세우는, 이 CP만 가능한 향의 서사."

AC'SCENT 03 x AC'SCENT 12

사용자 반응: "ㅋㅋㅋㅋ 미쳤다 이거 캡처해야 돼" (즉시 트위터 공유)
```

### 핵심 차이 요약

| 항목 | 기존 6타입 | 새 4대 케미향 |
|------|----------|-------------|
| 분류 축 | 관계의 성질 (형용사) | 서사의 패턴 (이야기) |
| 언어 | 학술적 (Resonance, Enigma) | 팬덤 일상어 (밀당, 달달, 슬로우번) |
| 범용성 | 어느 팬덤에서도 안 쓰는 단어 | 한/일/중/글로벌 모두 이미 쓰는 단어 |
| 고유성 | 6가지 중 하나 (반복) | 4범주 + 무한한 고유 칭호 |
| 향 연결 | 논리적 (같으면 같은 향) | 공감각적 (트로프의 감각이 향이 됨) |
| 공유 가치 | 낮음 (정보 카드) | 높음 (감성 카드 + 킬러카피) |
| 도감 수집 | 6개 (너무 적음/많음 어중간) | 4개 (명확) + 칭호 수집 (무한) |
